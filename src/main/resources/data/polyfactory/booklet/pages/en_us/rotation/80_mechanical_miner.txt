<yellow><item 'polyfactory:miner'></> is a rotational block that allows you to mine blocks and attack entities.
<nl2>
To do so, you need to first put a compatible tool into it. The damage and mining speed depends on the tool used.
Tools used by this block will get damaged. Mining/Attacking works the same as if player did it, including all side effects
and bonuses (like drops from mobs).
<nl>
This block can interact with blocks/entities that are at most 2 blocks away.
<nl>
Comparator signal generated by this block depends on tools durability.
<nl2>
To power this block you need to connect it on the back.
The stress generated by this machine depends on currently mined block and current tool.
When inactive it doesn't generate any stress.
