<yellow><item 'polyfactory:mixer'></> is a 1x2x1 item crafting machine block. It required rotational energy to work.
It has 6 input slots, 2 bucket worth fluid storage and 3 output slot. This block depends on the
<u><blue><pagelink '/basics/50_temperature'>Temperature System</*>.
<nl2>
In case of fluidless recipes, recipe output are always put into the output slots. For recipes that generate fluids,
the fluid output will be put into the same fluid storage as fluids used for inputs.
<nl>
For hopper/storage access, the input slots is loaded via front of the block. Input slots can be accessed by conveyors directly.
The output slot is accessible from the back. Conveyors can't access the output without a <item 'polyfactory:funnel'>
<nl2>
To power this block you need to connect it on the left or right side of the top part.
The stress generated by this machine depends on currently crafted recipe.
When inactive it doesn't generate any stress.
